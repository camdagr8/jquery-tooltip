Configuration


$("[data-toggle='tooltip']").tooltip({
    classes 	: "tooltip",
    content	: null,
    persistent 	: false,
    plainText 	: false,
    position 	: "top-center",
    target 	: "body",
    trigger 	: "hover"
});
Property Type Description Default
classes String A space delimited list of classes to apply to the tooltip container. "tooltip"
content String | $('.selector') Specifies the HTML to use inside the tooltip container. "parent"
persistent Boolean Specifies if the tooltip should hide when the opposite trigger is fired, a user clicks outside of the tooltip or when another tooltip is shown.
  • If the trigger is hover (mouseover), the tooltip will automatically hide when the mouseout event is fired.
  • If the trigger is focus, the tooltip will automatically hide when the blur event is fired.
  • If the trigger is click, the tooltip will automatically hide when the caller element is clicked again.
 false
plainText Boolean Whether to treat the [content] as plain text instead of HTML. false
position String
Where to position the tooltip.

Prefixing any position value with "auto-" (auto-left-middle) will dynamically position the tooltip. In cases where the tooltip will display off screen, the tooltip will be positioned in a position that allows it to be seen.

If the position value is "auto-left-middle" but will display off screen, "auto-right-middle" will be applied.

 top-center
target String | $('.selector') Appends the tooltip [content] to a specific element. "body"
trigger String
Event that will display the tooltip.

If the trigger is manual you will have to toggle the tooltip via the javascript methods [show | hide | toggle].

  • focus
  • hover
  • mouseover
  • manual
 "hover"

 

Methods

Method Params Description Returns
get property Gets the specified configuration property. String | Array
hide Hides the tooltip. Triggers the tooltip:hide event. Alias: 'close'.
redraw Redraws the tooltip. Triggers the tooltip:redraw event.
set property, value Sets the specified configuration property.
show Shows the tooltip. Triggers the tooltip:show event. Alias: 'open'.
toggle Hides or Shows the tooltip. Triggers the tooltip:toggle event. 

// Get an array of all tooltip targets
var tooltipTargets = $("[data-toggle='tooltip']").tooltip("get", "target");

// Set the first tooltip's trigger
$("[data-toggle='tooltip']:first").tooltip("set", "trigger", "focus"); 

// Hide the first tooltip
$("[data-toggle='tooltip']").tooltip("hide");

// Show the first tooltip
$("[data-toggle='tooltip']:first").tooltip("show"); 

// Toggle the first tooltip 
$("[data-toggle='tooltip']:first").tooltip("toggle");
